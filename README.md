# 📚 E-Library Management System

The **E-Library Management System** is a web application that helps users manage books in a digital library.
Users can **add, view, search, update, and delete books** easily through a simple and user-friendly interface.

## 🚀 Features

* 📖 Add new books to the library
* 🔍 Search books by title or author
* 📚 View all available books
* 🗑️ Delete books from the library
* 📱 Simple and responsive interface

## 🛠️ Technologies Used

### Frontend

* HTML
* CSS
* JavaScript

### Backend

* Python

### Framework

* Django

*  MySQL
